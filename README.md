## ✨ Features

- **Responsive Design** — works seamlessly on desktop, tablet, and mobile
- **Custom Cursor** — smooth animated cursor with ring effect
- **Scroll Animations** — elements reveal on scroll with staggered transitions
- **Animated Counters** — stats count up when they enter the viewport
- **Noise + Grid Background** — layered texture for a premium dark aesthetic
- **Floating Blobs** — ambient animated gradients in the hero section
- **Profile Image** — square photo with layered teal/blue shadow effects and corner brackets
- **Interactive Cards** — hover effects on skill, project, and experience cards

## 🚀 Getting Started Locally

No build step needed — just open the file directly:

```bash

git clone https://github.com/your-username/your-username.github.io.git

# Open in browser
open index.html
```

Or simply double-click `index.html` in your file explorer.

## 📸 Adding Your Profile Photo

1. Add your photo to the repo folder
2. In `index.html`, find this line:
   ```html
   <img src="your-photo.jpg" alt="Veeramallu Teja Sriram" />
   ```
3. Replace `your-photo.jpg` with your actual filename
4. Commit and push — it'll update live automatically
